Setup

Please be sure to properly adjust the database connection in the config file test_files/demoWebapp/res/Time_Tracker_Report.xml (at the end of the file.) After the configuration is corredt, please rebuild the demo war file and deploy it into the JSP container. In case you do not see any data in the reports, be sure to have data in your database. Demo data can be inserted into the database using the ant target fillDemoData , the database can be cleared using the ant target clearDatabase.

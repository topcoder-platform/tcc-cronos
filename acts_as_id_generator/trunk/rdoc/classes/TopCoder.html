Module TopCoder
In: lib/acts_as_id_generator.rb

This module provide a Rails plugin that will allow ActiveRecord models to easily generate their primary key ids from legacy Id Generator sequences.

Thread-safety: This component is not completely thread safe. But it can be used in thread-safe way. The IdGenerator and Errors have mutable properties, but they will not be set in multiple threads. The method next_id is synchronized which is the only method used in multiple threads, so this component can be used in thread-safe way.
